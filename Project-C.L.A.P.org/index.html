Project Overview

نظرة عامة على المشروع

Our team has developed an innovative device that allows users to control a fan with the sound of clapping. This project demonstrates the practical application of sound sensors and electronic circuits in creating easy-to-use automation solutions.

طور فريقنا جهازاً مبتكراً يتيح للمستخدمين التحكم في المروحة من خلال صوت التصفيق. يوضح هذا المشروع التطبيق العملي لمستشعرات الصوت والدوائر الإلكترونية في إنشاء حلول اوتوماتيكية سهلة الاستخدام.

How does it work?

كيف يعمل؟

The system operates by:

يعمل النظام من خلال:

  • Detecting sound waves via the microphone sensor
  • Processing the audio input with a microcontroller
  • Activating the relay to manage the fan's power state
  • Utilizing a debounce mechanism to avoid false triggers
  • كشف الموجات الصوتية عبر مستشعر الميكروفون
  • معالجة إدخال الصوت باستخدام المتحكم الدقيق
  • تشغيل الريلاي للتحكم في حالة طاقة المروحة
  • ستخدام آية منع الارتداد لتجنب التشغيل الخاطئ

Arduino Code

كود الأردوينو


const int soundSensorPin = 2;  // Sound sensor connected to digital pin 2
const int relayPin = 3;        // Relay module connected to digital pin 3
const int ledPin = 13;         // Built-in LED for status indication

boolean fanState = false;      // Current state of the fan
unsigned long lastClapTime = 0; // Time of the last detected clap
const int clapTimeout = 1000;  // Timeout between claps in milliseconds

void setup() {
    pinMode(soundSensorPin, INPUT);
    pinMode(relayPin, OUTPUT);
    pinMode(ledPin, OUTPUT);
    
    // Initialize with fan off
    digitalWrite(relayPin, LOW);
    digitalWrite(ledPin, LOW);
}

void loop() {
    // Read sound sensor
    int soundValue = digitalRead(soundSensorPin);
    
    // If sound detected (clap)
    if (soundValue == HIGH) {
        unsigned long currentTime = millis();
        
        // Check if enough time has passed since last clap
        if (currentTime - lastClapTime > clapTimeout) {
            // Toggle fan state
            fanState = !fanState;
            digitalWrite(relayPin, fanState);
            digitalWrite(ledPin, fanState);
            
            // Debounce delay
            delay(50);
        }
        lastClapTime = currentTime;
    }
}
